# FileCast

ブロックを並び替えて作成したルールに沿って柔軟にファイルのリネームを行えるツールです。

## 概要

- 「日付」「バージョン」といったブロックを並び替えて命名規則を作成することで、元のファイル名からこれらの要素を取得し、新たな配置で置き直してリネームすることができます。
- ブロックは、yaml をより柔軟にしたような DSL 形式でユーザーが自由に追加可能です。
- 「日付の形式」や、「区切り文字」などを自由に変更することが可能で、変更点はリアルタイムでサンプルに反映され、変更後のファイル名をイメージすることができます。

## DSL の解説
- ツール内でもカテゴリの編集画面の「記述方法リファレンス」から同じものが確認できます。

#### yaml記述方法
1. コロンの後に半角スペース1つが必要です……が、こっちで入れるので無くても動きます。

2. インデントは半角スペース2つ(または4つ)で行い、Tabは使用できません……が、このエディタではTabキーを押すとスペース2個が入力されるので、Tabも使えます。

3. リストを1行に書く場合、角括弧[]で囲う必要があります……が、こっちで入れるので無くても動きます。

    
#### 基本の書き方
```
カテゴリ名:
    アイテム名: 検索パターン, 検索パターン, ...
    アイテム名: 検索パターン, 検索パターン, ...
    ...
カテゴリ名:
    アイテム名: 検索パターン, 検索パターン, ...
    ...
```

と書くことで、検索パターンのいずれかに合致するものをアイテム名として配置します。


#### 便利機能
##### 1. 単一アイテムの略記
検索パターンとアイテム名が同じ場合、アイテム名を省略できます。

```
   # 以下はすべて同じ意味
   アイテム名: アイテム名
   アイテム名
   アイテム名:
   アイテム名::
```

##### 2. 同一アイテムの略記
検索パターンにアイテム名を含む場合、コロンを重ねることで、そのアイテム名を省略できます。

```
   # 以下は同じ意味
   アイテム名: アイテム名, 検索パターン, 検索パターン, ...
   アイテム名::  検索パターン, 検索パターン, ...
```

##### 3. 検索要件
カテゴリ内に、REQというアイテムを含めることで、REQ内のいずれかの単語が元のファイル名に含まれる場合にのみそのカテゴリの検索が有効になります。

```
Fruits:
    REQ: picture, image  # <- この行
    berries: strawberry, raspberry, blueberry
    citrus: orange, lemon
```

##### 4. 拡張子要件:
REQ 内をピリオドで始めると、拡張子がそのいずれかである場合にのみそのカテゴリの検索が有効になります。

```
    REQ: .png, .mp4
```

##### 5. 要件がないことの明示
REQ を 「*」のみにすると、常に検索が有効になります。これは、REQ の行を書かない場合と同じ意味になります。

```
    REQ: *
```

##### 6. 存在する場合のみ検索
REQ を 「!」のみにすると、検索パターンのいずれかに一致する場合のみ検索を有効にします。
通常、検索パターンが見つからなかった場合は、変更後のファイル名に No[Categry] のような文字列が含まれ、変換プレビューが Warning となりますが、
REQを「!」にしておくことで、存在しない場合は No[Categry] や Warning を出さずにパスします。

```
  REQ: !
```

    
#### 仕様詳細
1. 検索は大文字・小文字を無視して行います。

2. 検索対象は単語全体としてマッチするものに限ります。
    例えば、"education"内に"cat"は存在しませんが、
    "eduCatIon"や、"edu_Cat1on"内には存在する判定になります。

3. 日本語も使用できます。

4. (REQ内の「!」「*」を除く、)ファイル名に使用できない記号は使用できません。

5. REQ 内には、
    ```
        REQ: .png, .apple
    ```
    のように、拡張子と文字を混ぜて記載することができます。この場合もいずれか1つが存在すれば検索対象になります。

6. スペースやコロンなどの記号は、全角で入力されている場合半角に変換して処理します。

7. 一度保存して再び開くとテキストが整形されるため、テキストの見た目が変わる場合があります。
